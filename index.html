Semester Reflection

What I learned: By working through all six labs, I gained valuable experience in data visualization, critical analysis, and interactive mapping techniques.

  • Lab #1 was arguably the most challenging lab as it's task was to create this HTML and CSS website. As someone with zero prior experience with HTML and CSS, I found it overwhelming at first, but I persevered and learned a lot throughout the process, making me confident in my HTML and CSS skills.
  • Lab #2 was one of my favorite labs because I got to explore different AI creative tools, seeing how they could be beneficial, but alos seeing the flaws, both technical and ethical, within them as well.
  • Lab #3 was interesting because it allowed me to apply critical thinking skills and Tufte's principlesto evaluate existing visualizations not just for the information they portray, but for their effectiveness in communicating that information, which has and will continue to play a role in my work.
  • Lab #4 refreshed my knowledge and understanding of Tableau. I really enjoued this particular lab because it allowed me to deepen my understanding of the data visualization tools and techniques that are integral to becoming an engineering or data analyst, both of which are careers I am interested in.
  • In Lab #5, I used Vibe Coding to enhance my website. I found Vibe Coding to be a useful tool for improving the user experience and functionality of my website, especially since I did not have experience with web development prior to this course, showing the value of such a tool.
  • Finally, Lab #6 was a very personal lab that I enjoyed thorougly. I loved learning how to create an interactive map on MapBox and integretaing it into my website. But it isn't an arbitrary map, it's a map of some of my favorite places from my hometown, many of which were places I have fond childhood memories from.

Challenges I overcame: The greatest challenge I faced was learning the fundamentals of HTML and CSS. Because of my lack of experience, I was initially overwhelmed, but I persevered and learned a lot throughout the process. To combat this challenge, everytime I would generate code with AI or copy code from other sources, I made sure to understand what each line of code did and how it contributed to the overall functionality of the website by doing research and watching videos, which bettered my overall understanding of HTML and CSS.

How I grew: My understanding of digital culture and data analytics evolved as I learned how technology, data, and user interaction work together. Building my website with HTML and CSS helped me see how digital spaces are created and how design affects user experience. Evaluating visualizations using Tufte's principles and working with Tableau strengthened my ability to analyze and communicate data effectively. Additionally, exploring AI tools and Vibe Coding showed both the benefits and limitations of emerging technologies. Finally, creating an interactive MapBox map helped me understand how data and storytelling can come together in meaningful digital experiences. Overall, my skills and understanding of Digital Culture and Data Analytics have greatly improved.

Looking ahead: As I look ahead to my career in data analytics or engineering, I plan to use the skills I have learned in this course to analyze a given problem and find its solution using my coding HTML, CSS, and Python techniques. Additionally, I am confident that I will be able to convey my findings using the visualization techniques we have learned throughout the semester and during my time as a DCDA major at TCU.

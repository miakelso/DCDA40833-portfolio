Vibe Coding

Overview

In Lab 5, I completed two experiments using AI as a development assistant. Experiment A focused on enhancing my portfolio with a dark mode togglefeature, while Experiment B involved writing Python code to scrape data from a website. This lab emphasized understanding, not just generating, AI-assisted code.

Experiment A: Dark Mode Toggle Portfolio Enhancement

I used AI assistance to help design and debug a dark mode toggle for my portfolio. The feature allows users to switch between light and dark themes and saves their preference.

CSS Variables for Dark Mode
         
body.dark-mode {
    --text-color: #e0e0e0;
    --bg-color: #1a1a1a;
    --white: #2a2a2a;
}

I defined the site's main colors using CSS variables, and this block redefines them when dark mode is active, which automatically changes the colors across the site.

JavaScript Dark Mode Toggle

darkModeToggle.addEventListener('click', () => {
    body.classList.toggle('dark-mode');
    localStorage.setItem(
        'darkMode',
        body.classList.contains('dark-mode')
    );
});

I set up the site's colors using CSS variables, and I learned how an event listener can respond to a button click and switch those color values to change the site's appearance.

Experiment B: Python Web Scraping

In Experiment B, I used Python to scrape data from a website with help from AI. The AI helped me get started, and I focused on understanding how the code collected the data and what the output represented.

Core Web Scraping Logic

import requests
from bs4 import BeautifulSoup

response = requests.get(url)
soup = BeautifulSoup(response.text, 'html.parser')

This code shows how the scraper requests a webpage from the user and extracts information from it, even though AI helped generate the initial structure of the code.

Understanding Spectrum Reflection

  • Fully understood: How the dark mode toggle works visually, basic web scraping concepts, and the overall structure of the HTML and CSS.
  • AI-assisted but understood: Using JavaScript to respond to user interaction, dark mode persistence, and how different Python libraries work together to scrape a website.
  • AI-generated then modified: The initial dark mode toggle code and the Python scraper logic, which I adjusted to improve readability and export all scraped data into a CSV file.

Vibe Coding Guidelines

Throughout this lab, I treated AI as a support tool rather than a replacement for learning. My personal guidelines for using AI include:

  • Using AI to help generate or structure complex code, but not submitting it without reviewing and understanding it
  • Adjusting AI-generated code to better fit my project’s goals and requirements
  • Checking AI output carefully, especially when exporting or displaying data
  • Being transparent about AI use by noting when code contains AI-generated components
